# FlyRank Capstone Project

**A Frontend Engineering capstone project exploring AI-powered ranking and visualization.**

## Overview

FlyRank capstone project is an AI-driven application designed for the Frontend Engineering track in FlyRank AI program.

## Features

- Responsive frontend interface

## Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

```bash
git clone https://github.com/Felistus/flyRank-capstone-project.git
cd flyRank-capstone-project
npm install
```

### Usage

```bash
npm run dev
```
